## 🚀 About The Project

This is my personal portfolio website showcasing front-end development expertise, creative vision, and a passion for crafting immersive digital experiences. Built with cutting-edge web technologies, it features advanced animations, interactive 3D backgrounds, and a sleek, modern design.

### ✨ Key Features

- **Immersive 3D WebGL Background** – Real-time animated shader created with Three.js.
- **GSAP Animations** – Smooth scrolling, staggered text reveals, scroll-triggered effects, and magnetic buttons.
- **Interactive UI** – Custom cursor, tilt cards, liquid buttons, glitch text, and project hover effects.
- **Marquee Tech Stack** – Scrolling showcase of tools and technologies used.
- **Responsive Design** – Fully optimized for desktop, tablet, and mobile with Tailwind CSS.
- **Performance Optimized** – Lazy loading, minimal bundle size, and optimized shaders.
- **Animated Statistics** – Counter animations and skill bars that activate on scroll.

## 🛠️ Built With

- [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML) – Semantic structure
- [Tailwind CSS](https://tailwindcss.com/) – Utility-first styling
- [GSAP](https://greensock.com/gsap/) – ScrollTrigger, Flip, Draggable for animations
- [SplitType](https://github.com/louisguitton/split-type) – Character/word splitting for text animations
- [Font Awesome](https://fontawesome.com/) – Icons
- [Google Fonts](https://fonts.google.com/) – Playfair Display, Inter, JetBrains Mono
